Grading Rubric: Interactive Particle System

Course: CSW 38 - Web Animation

Instructor: Professor Pacheco

Stage 1: The Lone Particle (10 Points)

Criteria Points
Canvas Setup: HTML file includes a <canvas> element properly configured to fill the window. 2
Particle Object: A JavaScript object or class correctly represents the particle with all required properties (position, radius, color, velocity). 3
Animation Loop: requestAnimationFrame is used to create a loop that properly clears, updates, and draws the particle. 3
Edge Detection: Logic correctly detects collisions with all four canvas edges and reverses the appropriate velocity. 2

Stage 2: A Swarm of Particles (15 Points)

Criteria Points
Particle Array: An array is successfully implemented to store multiple particle objects. 3
Particle Generation: A function is written to generate a specified number of particles and populate the array. 5
Randomization: Particle properties (position, radius, color, velocity) are effectively randomized for variety. 5
Animation Loop Update: The animation loop correctly iterates through the particle array to update and draw all particles. 2

Stage 3: User Interaction and Polish (20 Points)

Criteria Points
Mouse Tracking: A mousemove event listener is implemented to accurately track and store the cursor's coordinates. 5
Interaction Logic: Particles correctly check their distance to the mouse and enlarge when the cursor is in proximity, returning to their original size when it is not. 10
Window Resizing: A resize event listener is added that correctly updates the canvas dimensions to match the browser window, ensuring the animation remains responsive. 5

Make sure to test your code thoroughly before submission!
