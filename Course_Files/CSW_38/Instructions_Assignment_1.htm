Canvas Animation Assignment

Instructions (17 pts. Total)

  1. After reading, reviewing, and learning the content and concepts in the W3 Schools Canvas assigned sections, (COMPLETE sections Intro through Canvas Text Color), proceed with the "Animation Assignment".
  2. Use the provided code page, Canvas-Assignment_1.htm, and UPDATE and CREATE your own version.
  3. You must make, at a minimum, the following changes:
    • a. Change the color of the moving object.
    • b. Change the location of the canvas in relation to the webpage layout.
    • c. Change the size of the object.
    • d. Attempt to change the direction of movement (e.g., vertical, diagonal).
    • e. Attempt to add a 2nd moving object.
    • f. Add your name in the Webpage near the top (e.g., in the header).
    • g. Rename the File before submitting / Publishing. Format to use is First initial.Last Name.Animation_1.htm (Example: T.Pacheco.Animation_1.htm)
    • h. Lastly, PUBLISH the page to your Hosting site and Submit the URL, web address.
  4. Your score will be assessed per the Rubric below.

Grading Rubric

Criteria Description Points
Student Name Student's full name is clearly visible near the top of the webpage. File name changed and Published online. 3
Object Color The color of the primary animated object has been successfully changed from the original. 4

Second Object A clear attempt was made to add a second, distinct animated object to the canvas. The object does not have to behave perfectly. 1
