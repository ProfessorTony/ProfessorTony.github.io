üíª Course Description: Your Mission, Should You Choose to Accept It...

Welcome to **Introduction to Computer Operating Systems**! Ever wonder how your laptop can stream music, download a file, and run a code editor all at the same time without crashing? Or how your phone keeps your private data safe while running dozens of apps? In this course we will go "under the hood" to explore the most complex piece of software ever created. An Operating System (OS) is the master coordinator‚Äîthe bridge between the physical circuitry of your computer and the software you use every day.


Official- Course Description:

     This course is an introduction to the inner workings of operating systems. It covers how memory is allocated in main memory and secondary memory, how programs are executed as processes and threads, multitasking and multiprocessing, networking, distributed systems, system performance, and security. Students will study various examples of an operating system such as Microsoft Windows, Apple iOS or the Google Android operating system. (UC, CSU)

üéØ Course Objectives

Upon successful completion of this course, the student will be able to:

  1. Describe the basic functions of an operating system.
  2. Compare and contrast different types of operating systems such as preemptive versus non-preemptive, centralized versus distributed or mobile versus desktop operating system.
  3. Describe how memories are allocated and managed by an operating system.
  4. Describe how programs, processes and threads are scheduled and managed by an operating system.
  5. Describe how storage memories are allocated and managed by an operating system.
  6. Describe how an operating system provides security to application software.
  7. Describe how an operating system recovers from errors that can happen when programs are running.
  8. Describe how an operating system recognizes a device, such as a printer, and then controls and manages the workings of that device.

üí° Learning Outcomes

Upon successful completion of this course, the student will be able to:

  1. Students will demonstrate an understanding of how a stand-alone computer or a device such as an iPhone functions correctly in the context of a public network and still keep its resources secure.
  2. Students will demonstrate an understanding of how an operating system interacts with users and manages its memory, its processes, its storage, and its various devices.
