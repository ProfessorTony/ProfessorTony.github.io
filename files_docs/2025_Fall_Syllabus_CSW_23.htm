üíª Course Description: Your Mission, Should You Choose to Accept It...

Welcome to Basic Web Design! Hey class! üëã Get ready to unleash your inner web designer! This course will transform you from a beginner into a web-building wizard. Learn the core languages‚ÄîHTML, CSS, and JavaScript‚Äîto create dynamic and beautiful websites. We'll explore everything from eye-catching images and videos to interactive forms and buttons. By the end, you'll have the skills to design, build, and launch websites that wow! üöÄ.

Official- Course Description:

     This course will provide a fundamental understanding of the methods and techniques of developing a simple to moderately complex website. Topics include: creating webpages with current standard webpage language (HTML), cascading style sheets (CSS), and Javascript. Exploration of incorporating images, audio/visual media and interactive tools like forms and image maps. This course prepares apprentice Web developers to identify the information needs of a client, design appropriate WWW solutions, and implement them. (HBCU, CSU)

üéØ Course Objectives

Upon successful completion of this course, the student will be able to:

  • Demonstrate the ability to create a HTML web page according to given specification and utilize tags for text, image, video and sound.
  • Demonstrate the ability to create a web page that will be responsive to different devices used by users to access the page using HTML, CSS and Javascript.

üí° Learning Outcomes

Upon successful completion of this course, the student will be able to:

  • Demonstrate the ability to create a simple static web page using basic HTML tags.
  • Demonstrate the ability to add multimedia files and modify appearance of a page using stylesheets (CSS) and XML.
  • Demonstrate ability to add responsive design features to a web page so that page can be viewed in multiple devices.
